Quick setup:

      cd
      mv .emacs.d dot-emacs-d.BAK
      git clone https://github.com/thekrampus/dot-emacs.git .emacs.d
      ln -s ~/.emacs.d/dot-emacs ~/.emacs
      echo "It's easy! Not hard!"
